1. 근사해
 - 현재 NP-Complete 문제의 모든 입력에 대한 최적해를 다항식 시간에 찾는 것은 불가능하다.
 - 어떤 방식으로든지 해결하려면 다음의 3가지 중에서 1가지는 포기해야 한다.
  -> 다항식 시간에 해를 찾는 것 
  -> 모든 입력에 대해 해를 찾는 것
  -> 최적해를 찾는 것
 - 보통 NP-Complete 문제를 해결하기 위해 최적해를 찾는 것을 포기한다.
 - 근사 알고리즘은 근사해를 찾는 대신에 다항식 시간의 복잡도를 가진다.
  -> 근사 알고리즘은 근사해가 최적해에 얼마나 근사한지를 나타내는 근사 비율(Approximation Ratio)을 제시해야 한다.
  -> 근사 비율은 근사해의 값과 최적해의 값의 비율로서, 1.0에 가까울수록 정확도가 높은 알고리즘이다.
  -> 최적해를 대신할 수 있는 '간접적인' 최적해로 근사 비율을 계산한다.

2. 모의 담금질 기법
 - 모의 담금질(Simulated Annealing) 기법은 높은 온도에서 액체 상태인 물질이 온도가 점차 낮아지면서 결정체로 변하는 과정을 모방해서 해를 탐색하는 알고리즘이다.
  -> 용융 상태에서는 물질의 분자가 자유로이 움직이는데 이를 모방하여, 해를 탐색하는 과정도 특정한 패턴 없이 이루어진다.
  -> 온도가 점점 낮아지면 분자의 움직임이 점점 줄어들어 결정체가 되는데, 해 탐색 과정도 이와 유사하게 점점 더 규칙적인 방식으로 이루어진다.
 - 모의 담금질 기법이 항상 전역 최적해를 찾아준다는 보장은 없다.
 * 모의 담금질 과정
  - 초기에 임의의 해 S를 선택한다.
  - 온도 T를 높은 값으로 설정한다.
  - 다음에 대해 반복한다.(온도 T 반영)
   -> S의 이웃해 중에 랜덤하게 하나의 해 S'를 선택
   -> S'이 S보다 좋은 해이면 S를 S'로 변경한다. S'이 S보다 나쁜 해이면 확률에 의해 S'을 S로 받아들일지 결졍한다.
   -> 온도 T를 감소시킨다.
  - 반복이 종료되면 S를 해로 선택한다.
